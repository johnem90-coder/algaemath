# AlgaeMath.com

Interactive educational platform for photobioreactor modeling.

**Tech Stack:** Next.js + TypeScript + Tailwind CSS + shadcn/ui + Recharts + KaTeX

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
